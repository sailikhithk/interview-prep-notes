2. Hebbian Learning — Synaptic Strengthening
Principle "Cells that fire together, wire together"

When two notes are co-accessed or one wikilinks to the other, their connection weight increases by delta=0.1 (capped at 1.0).

strength = (frequency × 0.4) + (recency × 0.3) + (success_rate × 0.3)

frequency × 0.4

How often the link is traversed

recency × 0.3

Exponential decay — recent = higher

success_rate × 0.3

% of times the link gave a useful result

connection_tracker.py
def strengthen_connection(self, source, target, delta=0.1):
    key = f"{source}--{target}"
    self.weights[key] = min(1.0, self.weights.get(key, 0.0) + delta)
    self._save()

def decay_connections(self, daily_rate=0.01):
    for k in self.weights:
        self.weights[k] *= (1 - daily_rate)   # w(t) = w0 * (1-r)^t
    self._save()

def prune_connections(self, threshold=0.1):
    self.weights = {k: v for k, v in self.weights.items() if v >= threshold}
    self._save()

Time: strengthen O(1) | decay O(E) | prune O(E)

5. KNN Similarity — Semantic Nearest Neighbours

Pre-computes cosine similarity between concept embedding vectors. Stores top-k neighbours in knn_graph.json for O(1) query time.

# O(N^2 * D) build — N concepts, D embedding dims
def update_knn_graph(self, k=5):
    for concept_id, emb in self.embeddings.items():
        sims = []
        for other_id, other_emb in self.embeddings.items():
            if other_id != concept_id:
                # cosine = dot(a,b) / (|a| * |b|)
                dot   = sum(x*y for x,y in zip(emb, other_emb))
                norm  = (sum(x**2 for x in emb)**0.5 * 
                         sum(x**2 for x in other_emb)**0.5 + 1e-10)
                sims.append((other_id, dot/norm))
        self.knn_graph[concept_id] = sorted(sims, key=lambda x:-x[1])[:k]
Build: O(N²·D) — run once, persist
Query: O(1) — JSON lookup
6. 3-Layer Retrieval — Ranked Merge
L1

Direct Search

Obsidian REST API full-text. Fastest. O(N) per query.

L2

Concept Keyword Index

Pre-built keyword→concept map from headings + bold text. O(1) lookup, catches synonyms.

L3

Graph Traversal

BFS depth=2 from matched concepts. Surfaces connected knowledge. O(V+E).

Merge & Rank: 
results = deduplicate(L1_hits + L2_hits + L3_hits)
ranked  = sorted(results, key=lambda r: (
    r.synaptic_weight * 0.4 +   # Hebbian strength
    r.recency_score   * 0.3 +   # time since last access
    r.direct_match    * 0.3     # exact keyword hit
), reverse=True)
